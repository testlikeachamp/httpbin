## Run the app locally

```
export FLASK_APP=httpbin/core.py
export FLASK_DEBUG=1
python3 -m flask run
```
By default the app will run on port 5000.
So just open browser and got to http://127.0.0.1:5000.
Or better hit the api with requests!
